# Alphabetter — Keybind Guide

All keybinds use the **Right Option (⌥)** key as the modifier. Hold Right ⌥ and press the listed key to insert the first symbol. **Press the same key again** (while still holding Right ⌥) to cycle through alternate symbols on that key.

Add **Shift (⇧)** while holding Right ⌥ for a second layer of symbols — mostly diacritics, tones, and secondary modifiers.

## Other Shortcuts

| Shortcut | Action |
|----------|--------|
| **Right ⌥ + Space** | Toggle the IPA palette window |

> **Tip:** Diacritics are combining characters — they attach to the preceding letter. Type your base character first, then use the keybind to add the diacritic.
